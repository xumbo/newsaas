## ✅ Checklist de Desenvolvimento

### Nova Feature
1. Criar branch feature/*
2. Desenvolver componentes
3. Testar localmente
4. Documentar alterações
5. Criar PR

### Deploy
1. Build local
2. Testes
3. Merge para main
4. Deploy automático
