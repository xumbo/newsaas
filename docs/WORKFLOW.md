# Workflow de Desenvolvimento

## 🌳 Branches
- `main`: Produção
- `feature/*`: Novas funcionalidades
- `fix/*`: Correções
- `docs/*`: Documentação

## 📝 Commits
Usar prefixos:
- ✨ `feat`: Nova feature
- 🐛 `fix`: Correção
- 📝 `docs`: Documentação
- 💄 `style`: Visual/UI
- ♻️ `refactor`: Refatoração
- 🚀 `deploy`: Deploy

## 🔄 Processo
1. Criar branch
2. Desenvolver
3. Testar
4. Documentar
5. PR
6. Review
7. Merge
